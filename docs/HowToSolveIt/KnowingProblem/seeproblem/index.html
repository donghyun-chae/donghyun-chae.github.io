    구하려는 것을 살펴봐 #

    내가 구하려는 것이 뭔지에 집중해 #

    답을 구하는 문제에선 구하려는 것이 뭔지 보고

    증명하는 문제에선 결론이 뭔지 봐.

    구하려는 것 살펴보기 #

    • 구하려는 것이 같은 다른 문제를 생각해봐. (ex 직선의 길이를 구할때 직선의 길이 구하는 다른 문제)

    • 전에 풀었던 문제를 떠올리면서 어떻게 활용할까? 보조 요소가 필요할까? 고민해봐.

    • 구하려는 것이 같거나 유사한 문제를 활용하면 문제에 접근하기 더욱 쉬워져.

    • 구하려는 것이 같은 문제가 없으면 문제가 상당히 어려워지는데 그럴땐 유사한 문제를 최대한 찾고 활용해.

    • 증명 문제 같은 경우엔 같은 결론을 갖는 문제라고 보면 돼.

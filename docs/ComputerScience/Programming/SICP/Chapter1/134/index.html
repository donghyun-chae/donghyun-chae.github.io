    SICP 1.34 #

    문제 #

    다음의 프로시저에서 (f f) 를 계산하려 하면 어떤 일이 벌어지고 왜 그럴까?

    #lang sicp
(define (f g)
  (g 2))

(define (square x)
  (* x x))

(f square)

(f (lambda (z) (* z (+ z 1))))

(f f)


    문제풀이 #

    (f f) 의 실행 결과는 다음과 같다.

    그 이유는 (f g) 에서 받는 g는 프로시저를 인자로 받는 자리이다.
    (g 2)에서 프로시저에 쓰이기 때문에 프로시저가 아니면 arguments does not match 이런 문구를 포함한 오류가 난다.

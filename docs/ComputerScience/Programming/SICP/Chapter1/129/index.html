    SICP 1.29 #

    문제 #

    심프슨 방법은 정적분 값을 더 정확히 근사하는 방법이다.
    심프슨 방법에서 정적분 값은 \(\frac{h}{3}(y_0+4y_1+2y_2+4y_3+...+2y_{n-2}+4y_{n-1}+y_n)\) 이렇게 구할 수 있으며 짝수 n에 대하여 \( h=\frac{b-a}{n} \) 이고 \(y_k=f(a+kh)\) 이다.
    f,a,b,n 을 인자로 받는 심프슨 방법 프로시저를 작성하고 기존의 방법과 비교해봐라.



    문제풀이 #

    심프슨 방법에서 첫 항과 끝 항을 제외하고 홀수항은 4, 짝수항은 2의 계수를 갖는다.
    따라서 짝수 항을 짝수 항끼리 홀수 항을 홀수 항끼리 묶을 수 있다.
    \[\frac{h}{3}(y_0+4(y_1+y_3+...+y_{n-1})+2(y_2+...+y_{n-2})+y_n)\] 따라서 \(y_0+y_n\) 을 따로 더해주고 4를 곱한 홀수 항과 2를 곱한 짝수항을 더해준 후 3/h를 곱하면 된다.

    #lang sicp
(define (integral f a b dx)
  (define (add-dx x) (+ x dx))
  (* (sum f (+ a (/ dx 2.0)) add-dx b) dx))

(define (sum term a next b)
  (if (> a b)
      0
      (+ (term a)
         (sum term (next a) next b))))

(define (cube x)
  (* x x x))

(define (simpson f a b n)
  (define h (/ (- b a) n))
  (define (plus-2h x) (+ x h h))
  (* (+ (f a)
        (* 2 (sum f (plus-2h a) plus-2h (- b h h)))
        (* 4 (sum f (+ a h) plus-2h (- b h)))
        (f b))
     (/ h 3)))

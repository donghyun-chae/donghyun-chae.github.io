    SICP 1.28 #

    문제 #

    페르마 검사가 틀리지 않게 바꾼 검사 중 밀러-라빈 검사가 있다.
    페르마 검사와 달리 \(n\) 이 소수고 \( 0 < a < n \) 인 정수 \(a\) 에 대하여 \(a^{n-1} \pmod n = 1\) 로 맞아떨어진다.
    밀러-라빈 검사 expmod 에서 제곱을 할 때마다 ‘ \( 1 \pmod n\) 의 뻔하지 않은 제곱근’, 즉 1이나 \(n-1\) 은 아니면서 제곱한 값이 \( 1 \pmod n\) 인 수인지 알아보고 그런 제곱근이 있다면 \(n\) 이 소수가 아님을 밝힐 수 있다.
    위와 같은 내용을 참고하여 밀러-라빈 검사 프로시저를 작성하라.



    문제풀이 #

    이 문제의 핵심인 ‘ \( 1 \pmod n\) 의 뻔하지 않은 제곱근’은 \(n\) 이 소수라면 \( a^2 \pmod n = 1\) 을 만족시키는 \(a\) 의 값은 1이거나 \(n-1\) 밖에 없다.
    그도 그럴 것이 \( a^2 \pmod n = 1\) 를 달리 써보면 \( a^2 \pmod n = 1\) 이고 \( a^2 = nk+1 \) (k는 n 보다 작은 임의의 상수)이다.
    \( a^2 - 1 = nk \) 이고 \( (a+1)(a-1) = nk \) 일때 n 은 k보다 크기에 \( n = (a+1) \) 이고 \( a = n-1 \) 이기에 n 이 소수일 때 a는 1 혹은 n-1 이어야 위의 식을 만족시킬 수 있다.

    #lang sicp
(define (square x)
  (* x x))

(define (mr-test n)
  (define (try-it a)
    (= (expmod a (- n 1) n) 1))
  (try-it (+ 1 (random (- n 1)))))
    

(define (expmod base exp m)
  (cond ((= exp 0) 1)
        ((even? exp)
         (remainder (if (obvious-prime? (square (expmod base (/ exp 2) m)) m)
                        0
                        (square (expmod base (/ exp 2) m)))
                        m))
        (else
         (remainder (* base (expmod base (- exp 1) m)) m))))

(define (obvious-prime? n m)
  (if (not (or (= n 1) (= n (- m 1))))
      (= n (remainder 1 m))))

    검토 #

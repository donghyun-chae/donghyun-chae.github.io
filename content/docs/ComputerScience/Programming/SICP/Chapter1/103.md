# SICP 1.03

## **문제**
{{< hint info >}}
숫자 3개를 받아 큰 숫자 2개를 제곱한 후 그 2개 값을 덧셈하는 프로시저 정의하기.
{{< /hint >}}

---
## **문제풀이**
{{< hint warning >}}
```scheme
#lang scheme
(define (sum_large_square x y z)
  (cond
    ((and (>= x z) (>= y z))
     (+ (* x x) (* y y)))
    ((and (>= y x) (>= z x))
     (+ (* y y) (* z z)))
    ((and (>= z y) (>= x y))
     (+ (* z z) (* x x)))
  )
)
```

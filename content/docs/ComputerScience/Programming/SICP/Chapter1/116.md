# SICP 1.16

## **문제**
{{< hint info >}}
```scheme
#lang sicp
(define (even? n)
  (= (remainder n 2) 0))

(define (square x) (* x x))

(define (fast-expt b n)
  (cond ((= n 0) 1)
        ((even? n) (square (fast-expt b (/ n 2))))
        (else (* b (fast-expt b (- n 1))))))
```

### **현재의 fast-expt 는 로그 비례로 계산 시간이 늘어나는데 이를 유지하며 되도는 프로세스에서 반복하는 프로세스로 바꿔보자**
tip: 프로세스가 처음 돌아갈때 새 변수 a는 1로 시작해서 끝날때 결과값이 들어있게 한다. 상태가 바뀌어도 변하지 않는 것을 찾아보자.
{{< /hint >}}

## **문제풀이**
{{< hint warning >}}
```scheme
#lang sicp
(define (even? n)
  (= (remainder n 2) 0))

(define (square x) (* x x))

(define (fast-expt b n)
  (define (iter a b n)
    (cond ((= n 0) a)
          ((even? n) (iter a (square b) (/ n 2)))
          (else (iter (* a b) b (- n 1)))))
  (iter 1 b n)
  )
```
</br>
다음과 같은 핵심 아이디어를 활용하면 된다.
{{< katex display=true >}} b^n = (b^2)^{n/2} = (b^{n/2})^2 {{< /katex >}}
b를 제곱함으로써 지수를 뜻하는 n 을 반으로 줄일 수 있다.

# SICP 1.19

## **문제**
{{< hint info >}}
```scheme
#lang sicp
(define (square x)
  (* x x))

(define (fib n)
  (fib-iter 1 0 0 1 n))

(define (fib-iter a b p q count)
  (cond ((= count 0) b)
        ((even? count)
         (fib-iter a
                   b
                   ??
                   ??
                   (/ count 2)))
        (else (fib-iter (+ (* b q) (* a q) (* a p))
                        (+ (* b p) (* a q))
                        p
                        q
                        (- count 1)))))
```
1. Tpq 는 (a,b)를 두고 a <- bq+aq+ap, b <- bq+aq 라는 규칙을 나타낸다고 할 때 Tpq 를 두번 거듭 계산한 값이 Tp'q' 을 한 번 한 것과 같다는 것을 밝히고, p와 q로 p'과 q'을 계산하는 식을 밝혀라.   

2. 위 프로시저에서 ?? 부분을 채워보자.   
tip: 이 알고리즘은 제곱 알고리즘을 효율화 할 때 쓰인 아이디어와 같은 아이디어이다.
{{< /hint >}}

## **문제풀이**
{{< hint warning >}}

1. T = (a+b, a) 라 할때 Tpq = (bq+aq+ap, bp+aq) 이다. 이를 통해 계산하면 다음과 같다.   
{{< katex display=true >}} T_{pq}^2 = (bp+aq)q + (bq+aq+ap)q + (bq+aq+ap)p, (bp+aq)p + (bq+aq+ap)q \\ = b(q^2+2pq)+a(2q^2+2pq+p^2), b(p^2+q^2)+a(q^2+2pq) {{< /katex >}}

위의 정리된 식을 살펴보면 Tpq 을 2번 거듭한 것 모양에 Tpq 가 있는 것을 발견 할 수 있다.      
따라서 p', q' 을 나타내면 p' = (p^2+q^2) 이고 q' = (q^2+2pq) 이다.   
</br>

2. 
```scheme
#lang sicp
(define (square x)
  (* x x))

(define (fib n)
  (fib-iter 1 0 0 1 n))

(define (fib-iter a b p q count)
  (cond ((= count 0) b)
        ((even? count)
         (fib-iter a
                   b
                   (+ (square p) (square q))
                   (+ (square q) (* 2 p q))
                   (/ count 2)))
        (else (fib-iter (+ (* b q) (* a q) (* a p))
                        (+ (* b p) (* a q))
                        p
                        q
                        (- count 1)))))
```

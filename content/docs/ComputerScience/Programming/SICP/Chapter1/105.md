# SICP 1.05 

## **문제**
{{< hint info >}}
인자 먼저 계산법 vs 정의대로 계산법
```scheme
(define (p) (p))
(define (test x y)
  (if (= x 0)
      0
      y))

```
{{< /hint >}}

---
## **문제풀이**
{{< hint warning >}}
**(test 0 (p))**

**인자 먼저 계산법**
1. (test 0 (p))
2. (test 0 (p))   
...   
3. ∞(test 0 (p))   

**정의대로 계산법**
1. (test 0 (p))
2. (if (= 0 0)   
        0   
        (p))   
3. 0

인자 먼저 계산법은 인자가 먼저 평가 되기 때문에 (define (p) (p)) 자기 자신을 호출하는 (p) 로 인해 무한 루프에 빠진다.   
정의대로 계산법은 우선 계산을 펼칠 수 있을 때 까지 펼치기 때문에 (p) 를 호출하기 전에 if 가 먼저 실행된다.
{{< /hint >}}

# SICP 1.05 

## **문제**
>실행기가 인자 먼저 계산법을 따르는지 정의대로 계산법을 따르는지 알아보기 위해 다음과 같이 두 프로시저를 정의했을 때 어떤 결과가 나오는지 밝혀라.
```scheme
(define (p) (p))
(define (test x y)
  (if (= x 0)
      0
      y))

(test 0 (p))
```

---

</br>
</br>

## **문제풀이**
>인자 먼저 계산법
```
(test 0 (p))
(test 0 (p))
...
```
  

>정의대로 계산법
```
(test 0 (p))
(if (= 0 0) 
        0   
        (p))
0
```

>인자 먼저 계산법은 인자가 먼저 평가 되기 때문에 (define (p) (p)) 자기 자신을 호출하는   
>(p) 로 인해 무한 루프에 빠진다.   
정의대로 계산법은 우선 계산을 펼칠 수 있을 때 까지 펼치기 때문에 (p) 를 호출하기 전에 if 가 먼저 실행된다.

# SICP 1.12

## **문제**
{{< hint info >}}
파스칼 삼각형을 되도는 프로세스가 나오게끔 프로시저를 작성하라.

{{< /hint >}}


---
## **문제풀이**
{{< hint warning >}}
## 파스칼 삼각형
```scheme
#lang scheme
(define (p-recursion index n)
  (cond ((= index 1) 1)
        ((= index n) 1)
        ((= n 1) 1)
        (else (+ (p-recursion (- index 1) (- n 1))
                 (p-recursion index (- n 1))))))

(define (pascal n)
  (define (p-display index n)
    (cond ((< index n) (p-display (+ 1 index) n)))
   (display (p-recursion index n))
   (display " "))
  (p-display 1 n))

(define (pascal-all n)
  (cond ((> n 0) (pascal-all (- n 1))))
  (pascal n)
  (newline)
  )

```

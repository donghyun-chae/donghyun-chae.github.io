# SICP 1.15

## **문제**
{{< hint info >}}
```scheme
#lang sicp
(define (cube x) (* x x x))

(define (p x) (- (* 3 x) (* 4 (cube x))))

(define (sine angle)
  (if (not (> (abs angle) 0.1))
      angle
      (p (sine (/ angle 3.0)))))
```
라디안 각 x 로 sinx 를 구하는 프로시저이다.

### **1. (sine 12.15) 를 구할 때 p 프로시저가 호출되는 횟수는 몇 번인가?**
### **2. (sine a) 값을 계산할때 프로시저가 만들어내는 프로세스의 기억 공간과 계산 단계의 자람 차수를 a의 함수로 나타내면?**

## **문제풀이**
{{< hint warning >}}
### **1. (sine 12.15) 를 구할 때 p 프로시저가 호출되는 횟수는 몇 번인가?**
```
(sine 12.15)
(p (sine (/ 12.15 3.0)))
(p (p (sine (/ 4.05 3.0))))
(p (p (p (sine (/ 1.35 3.0)))))
(p (p (p (p (sine (/ 0.45 3.0))))))
(p (p (p (p (p (sine (/ 0.15 3.0)))))))
(p (p (p (p (p (p (sine (/ 0.05 3.0))))))))
(p (p (p (p (p (p 0.05))))))
```
위와 같이 프로세스가 형성되기 때문에 6번 호출된다.


### **2. (sine a) 값을 계산할때 프로시저가 만들어내는 프로세스의 기억 공간과 계산 단계의 자람 차수를 a의 함수로 나타내면?**

위 프로시저에서 기억 공간과 계산 단계 모두 a 가 3배 늘어날 때마다 1씩 증가한다.   
즉 log3(a) 만큼 증가하는 것으로 빅-오 표기법으로는 O(log(n)) 과 같다.

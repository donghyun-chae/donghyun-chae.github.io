# **결과를 점검할 수 있어?**

{{< hint normal >}}

가장 먼저 우리의 직관으로 결과를 점검하는거야.

만약 어떤 할아버지의 나이를 구하는데 8살 이라는 결과라면 직관적으로 틀렸음을 알 수 있지.

수치로 주어진 문제보다는 문자로 주어진 문제가 더 쉽게 접근할 수 있기에 적절한 변형이 필요할 때가 있어.

논증에서의 점검은 취약점 검토와 합리적 추론으로 진행돼.

{{< /hint >}}

# **구하려는 것이 뭐야?**

## 문제를 보며 **구하려는것**이 뭔지 스스로에게 물어봐.

{{< hint info >}}

구하려는 것이 무엇인지 부터 무엇이 주어져있고 어떤 조건이 있는지 같은 질문을 스스로에게 해봐.

이런 질문이 너가 문제에 대해 얼마나 **이해했는지** 알려줄꺼야.

문제에 따라서는 구하려는 것이 여러 개 있을 수 있고 조건을 따로 생각해야 할 수도 있고 경우가 다양해.

그니깐 질문을 조금 바꿔서 처음 자료는 뭐지? 두번째는? 문제에 따라서 질문을 **변형**해봐.

**증명하는 문제**에선 중요한 것이 가정과 결론이라 무엇을 가정하고 있어? 같은 질문이 유용해.
{{< /hint >}}

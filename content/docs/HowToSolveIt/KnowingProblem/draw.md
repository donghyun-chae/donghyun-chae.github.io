# **그림으로 그려봐**

### 그림으로 그려보는 것은 기하 문제가 아니더라도 중요한 도움을 주는 경우가 많아.

{{< hint info >}}

기하 문제는 세세한 것들을 검토하는데 **큰 도움**을 얻을 수 있어.

그림을 그리는 것은 **어떤 가정을 하는 것**이며 구하려는 것과 자료를 포함하는 가정은 필요해.

그림을 너무 정확하게 그릴려고 할 필요는 없지만 **의미는 정확하게** 전달되게 그려야해.

그림은 **도움**의 존재이지 <mark>근거</mark>가 아니야.

{{< /hint >}}

# **자료에서 유용한 것을 찾아보자**

### 구하려는 것과 자료에 징검 다리를 놓아보자.

### 다리의 시작은 때로는 구하려는 것으로 부터 때로는 자료로 부터 시작돼.

{{< hint warning >}}

보통은 구하려는 것으로 부터 생각을 시작하는 것이 좋지만 자료로 부터 시작하는 것도 가끔 유용해.

예를 들어 세 점 A, B, C 가 주어졌을 때 B 와 C 사이 정확히 지나는 A를 통과하는 직선을 그린다는 문제라면

**구하려는 것**은? <mark>직선</mark>

**자료**는? <mark>점 A,B,C 의 위치</mark>

**조건**은? <mark>A 를 통과하면서 B,C 로 부터 같은 거리이고 사이를 지나야한다.</mark>

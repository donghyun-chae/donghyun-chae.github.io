# **관련된 문제를 풀어봤으면 그것을 활용할 수 있을까?**

### 전에 풀어본 적이 있고 현재 풀려는 문제와 관련이 있다면 아주 좋아.

### 여기서 얘기하는 것은 상당히 자주 접하는 상황이고 중요해.

{{< hint warning >}}

**A** 라는 문제를 풀기 위해 **B** 라는 문제를 생각해냈다고 하자.

**B** 문제를 생각해 냈지만 2가지 종류의 문제가 있을 수 있어.

- 1. **B** 문제를 어떻게 사용할지는 알겠는데 그 해결 방법을 모르겠는 경우

- 2. **B** 문제를 어떻게 활용할 지 모르는 경우
     {{< /hint >}}

# **조건을 여러 부분으로 나눠봐**

{{< hint info >}}

문제에서 구해야하는 것, 자료, 조건을 따로 생각하는데 아이디어가 안떠오르면 **작게 나눠봐**.

자료도 따로 따로 생각해보고 전체적인 조건을 이해했다면 조건을 여러 부분으로 **분해**해봐.

이렇게 주어진 것들을 **분리하고 다시 결합**하는 것은 문제를 명확히 아는 것에 도움을 줘.

# **결의, 희망, 성공**

### 문제를 해결하는 것은 지적인 활동을 넘어서 **의지와 정서가 중요하다.**

<hr/>

{{< hint normal >}}
**_1._** 잘될 때 계속하는 것은 쉽지만 <span style = 'background-color:#F7DDBE'>**일이 잘 안풀릴때 견디기는 어렵다.**</span>  
계속하기 위해선 <span style = 'background-color:#D6F0FF'>**희망을 갖고 출발**</span>해야 하며, <span style = 'background-color:#D6F0FF'>**작은 성공을 반복해야한다.**</span>  
성공하지 못했다고 경멸해서는 안되며 <span style = 'background-color:#D6F0FF'>**성공할 수 있는 목표**</span>를 찾아야 한다.

<br/>
<br/>

**_2._** 문제를 해결하는 것은 <span style = 'background-color:#F7DDBE'>**의지의 문제다.**</span>  
문제를 해결하고자 하는 욕구가 없다면 아무런 문제도 해결 할 수 없다.  
<span style = 'background-color:#D6F0FF'>**실패를 통해 인내를 배우고**</span> <span style = 'background-color:#fff5b1'>**조그만 진전의 가치를 평가하는 법을 배우며**</span>  
<span style = 'background-color:#F7DDBE'>**핵심적인 아이디어를 기다리는 것을 배우고 그것이 나타났을 때 모든 힘을 집중하는 법을 배운다.**</span>

{{< /hint >}}
